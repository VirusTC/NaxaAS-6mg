1\. Structured ICD-10-CM Coding Matrix

This matrix maps secondary conditions and systemic clinical presentations directly to primary diagnostic codes, ensuring full alignment with standard institutional billing guidelines.

| Presentation / Symptom | Primary ICD-10-CM Code | Secondary / Manifestation Code | Clinical Documentation Requirement |
| **Primary Copper Deficiency** | **E83.09** (Other disorders of copper metabolism) | N/A | Use for documented nutritional or acquired hypocupremia. |
| **Copper Overload Monitoring** | **E83.01** (Wilson's disease / Copper Overload) | N/A | Apply when tracking potential accumulation during high-dose active repletion. |
| **Systemic Fungal Manifestation** | **B49** (Unspecified mycosis) | N/A | Primary documentation code for target fungal or mold-related treatments. |
| **Hypocupremic Anemia** | **E83.09** (Primary cause) | **D63.8** (Anemia in other chronic diseases classified elsewhere) | Both codes must be linked; do not report D63.8 alone. |
| **Hypocupremic Neutropenia** | **D70.9** (Neutropenia, unspecified) | **E83.09** (Other disorders of copper metabolism) | Document absolute neutrophil count (ANC) \(< 1500/\mu\text{L}\). |
| **Copper-Deficiency Myelopathy** | **G32.0** (Subacute combined degeneration of spinal cord in diseases classified elsewhere) | **E83.09** (Other disorders of copper metabolism) | Explicitly document neurological deficit mapping to posterior columns. |
| **Optic Neuropathy** | **H47.20** (Unspecified optic atrophy) | **E83.09** (Other disorders of copper metabolism) | Note central scotomas or prolonged visual evoked potentials (VEP). |
