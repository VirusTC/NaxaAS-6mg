[ ] STEP 1: HEMATOLOGICAL PHENOTYPING

    [ ] Draw Complete Blood Count (CBC) with differential.

    [ ] Identify macrocytic anemia (MCV > 100 fL) OR microcytic anemia.

    [ ] Verify absolute neutropenia (ANC < 1500 cells/µL).

    * clinical Note: Both B12 and Copper deficiency cause macrocytic anemia and neutropenia. 

      Bone marrow biopsy for copper deficiency unique shows ringed sideroblasts and cytoplasmic vacuolation.

[ ] STEP 2: METABOLIC CLEARANCE (RULE OUT VITAMIN B12 DEFICIENCY)

    [ ] Measure Serum Vitamin B12 levels.

    [ ] Measure Serum Methylmalonic Acid (MMA) levels.

    [ ] Measure Total Homocysteine levels.

    * Decision Rule: 

      - If B12 is Low (< 200 pg/mL) AND MMA is High -> Diagnosed Vitamin B12 Deficiency.

      - If B12 is Normal AND MMA is Normal -> Proceed to Copper Panel.

[ ] STEP 3: THERAPEUTIC COPPER PANEL COLLECTION

    [ ] Measure Total Serum Copper (Target Deficiency: < 70 µg/dL).

    [ ] Measure Serum Ceruloplasmin (Target Deficiency: < 20 mg/dL).

    [ ] Measure 24-Hour Urinary Copper Excretion.

[ ] STEP 4: EXOGENOUS INTERFERENCE EVALUATION (RULE OUT ZINC OVERLOAD)

    [ ] Measure Plasma/Serum Zinc levels (Reference range: 70–120 µg/dL).

    [ ] Interview patient regarding active use of:

        [ ] High-dose zinc supplements (> 40–50 mg/day)

        [ ] Zinc-containing denture adhesive creams

        [ ] Cold remedies / lozenges used long-term

    * Clinical Note: Excess zinc induces intestinal metallothionein production. 

      Metallothionein preferentially binds copper within enterocytes, preventing system absorption

      and shedding it in feces. If zinc is elevated (> 150 µg/dL), zinc cessation must accompany 
      cupric therapy.

[ ] STEP 5: REGULATORY & REPLESTION SIGN-OFF

    [ ] Verify baseline AST/ALT are within 1.5x of normal limits.

    [ ] Secure executed Voluntary Patient Disclosure & Informed Consent Form.

    [ ] Initiate single-patient expanded access documentation (Form FDA 3926) if applicable.
